重要なHTMLタグ一覧

このページは、Railsチュートリアル 「HTML編」で多用されるタグのいくつかをすぐ参照できるクイックリファレンスページと して作られます。このページの制作を通じて、HTMLのテーブル（table）をtable タグやそれに関連するタグの使い方を学びます。

以下のテーブルに含まれるHTMLタグは全体の一部に過ぎませんが、重要なHTMLについてはここで参照できます。

ブロックレベルタグ

HTMLタグ タグ名 目的
h1–h6 見出し（heading） 見出し文を囲む（レベル1〜6）
p パラグラフ（paragraph） テキストのパラグラフを囲む
table テーブル 表を囲む
tr テーブル行（table row） 行データを囲む
th テーブルヘッダー（table header） テーブルヘッダーを記述する
td テーブルデータ（table data） テーブルデータのセルを囲む
div ディビジョン（division: 区画） ドキュメント内にブロックレベルのセクションを定義する
header ヘッダー ページのヘッダーを表示する
ol 順序付きリスト（ordered list） 番号順に表示されるリスト要素
ul 順序なしリスト（unordered list） 順序が重要でないリスト要素
li リストの項目（list item） リスト項目を囲む（olとulのどちらにも使う）
blockquote ブロッククォート（block quotation） フォーマット済み引用表示
br 改行（break） 改行する

インラインタグ

HTMLタグ タグ名 目的 記述例 画面上の表示結果
em 軽い強調（emphasize） テキストを軽く強調する <em>技術の成熟</em> 技術の成熟
strong 強調（strong） テキストを強調する <strong>少なくとも10億人</strong> 少なくとも10億人
a アンカー（anchor） ハイパーリンクを作成する <a href="#">Learn Enough</a> Learn Enough
img 画像（image） 画像を囲む <img src="mhartl.jpg" alt="Michael Hartl">
code コード ソースコードとして整形する <code>table<code> table
span スパン ドキュメントのインラインセクションを定義する <span>イシュマエルと呼んでくれ。</span> イシュマエルと呼んでくれ。

ドキュメントタグ

HTMLタグ タグ名 目的
html HTML その文書がHTML文書であることを指定する
head ヘッダ（HEAD） 文書のヘッダ部分を指定する
body BODY ブラウザの画面上に表示される内容を指定する
title タイトル（title） 文書にタイトルをつける
meta メタ情報（META-information） 文書のメタ情報を指定する
